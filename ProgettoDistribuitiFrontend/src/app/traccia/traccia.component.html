Traccia:

Implementare un sistema distribuito per l’estrazione del sentiment a partire da dati social.

Il sistema deve consentire la sottomissione di dataset da parte degli utenti.
Successivamente, sfruttando le API messe a disposizione da:
● Servizi cognitivi di Azure

Per ogni dataset elaborato, il sistema deve dare la possibilità all’utente di recuperare il risultato,
ottenendo anche dei grafici che visualizzano la distribuzione del sentiment tra le diverse classi considerate (neutro, positivo, negativo, ecc.).

Per lo svolgimento del progetto, lo studente potrà utilizzare dei dataset di esempio scaricabili al link:
